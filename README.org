** Get
*** Dependancies
- =makey.el= (If you're using Magit, =makey.el= should have been already installed.)

*** Installation
Download or clone repository to your computer, then add these:
#+BEGIN_SRC lisp
(add-to-list 'load-path "/PATH/TO/fsc/")
(require 'makey)
(require 'fsc)
(global-set-key (kbd "C-c f") 'fsc)
#+END_SRC
